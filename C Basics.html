Understanding C Programming

Learn syntax, variables, data types, and operators in C with clear explanations and practical examples.

1. Basic Syntax of C

C programs follow a specific structure. Every C program must have a main() function where execution starts.

Example Program


#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}
  • #include <stdio.h> → Includes input/output functions
  • int main() → Entry point of the program
  • printf() → Displays output
  • return 0; → Ends program execution

2. Variables in C

A variable is a container used to store data values. Each variable must be declared with a data type.

Syntax


data_type variable_name;

Example


int age = 20;
float height = 5.9;
char grade = 'A';
  • int → Stores integers
  • float → Stores decimal values
  • char → Stores single characters

3. Data Types in C

Primary Data Types

  • int → Integer values
  • float → Decimal numbers
  • double → Large decimal numbers
  • char → Single character

Example


int count = 10;
float price = 99.99;
double pi = 3.141592;
char letter = 'C';

4. Operators in C

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus

int a = 10, b = 3;
printf("%d", a + b); // 13

Relational Operators

  • == Equal to
  • != Not equal
  • > Greater than
  • < Less than

Logical Operators

  • && Logical AND
  • || Logical OR
  • ! Logical NOT

int x = 5, y = 10;
if (x < y && y > 0) {
    printf("Condition is true");
}
