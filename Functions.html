Functions and Strings in C Programming

Detailed Explanation with Syntax and Examples

1. Functions in C

What is a Function?

A function in C is a self-contained block of code that performs a specific task. Functions help in code reusability, modularity, and easy debugging.

Why Functions are Used?

  • Reduce code repetition
  • Improve readability
  • Easy testing and maintenance
  • Divide large programs into small parts

Types of Functions

  • Library Functions – printf(), scanf(), strlen()
  • User-defined Functions

Syntax of Function

return_type function_name(parameters) { // function body return value; }

Example of User-Defined Function

#include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int main() {
    int sum = add(10, 20);
    printf("Sum = %d", sum);
    return 0;
}

2. Strings in C

What is a String?

A string in C is an array of characters that ends with a null character '\0'.

String Declaration

char str[size];
char name[20];

String Initialization

char city[] = "New York";

String Input & Output

char name[20];
scanf("%s", name);
printf("%s", name);

3. String Functions in C

strlen()

Returns length of string

strlen(string);
printf("%d", strlen("Hello"));

strcpy()

Copies one string to another

strcpy(dest, src);
char a[20];
strcpy(a, "C Language");

strncpy()

strncpy(dest, src, n);
strncpy(a, "Programming", 5);

strcat()

Joins two strings

strcat(str1, str2);
char a[20] = "Hello ";
strcat(a, "World");

strncat()

strncat(str1, str2, n);
strncat(a, "World", 3);

strcmp()

Compares two strings

strcmp(str1, str2);
if(strcmp("abc","abc")==0)
    printf("Equal");

strncmp()

strncmp(str1, str2, n);
strncmp("Hello","Help",3);

strchr()

Finds first occurrence of character

strchr(str, ch);
strchr("Hello",'e');

strrchr()

Finds last occurrence of character

strrchr(str, ch);
strrchr("Hello",'l');

strstr()

Finds substring

strstr(str, sub);
strstr("Programming","gram");

strlwr(), strupr(), strrev()

Case conversion and reverse (compiler dependent)

strlwr(name);
strupr(name);
strrev(name);
