Arrays and Pointers in C

Detailed explanation of arrays and pointers in C programming with syntax and multiple examples.

1. Arrays in C

An array is a collection of elements of the same data type stored in contiguous memory locations. It allows storing multiple values using a single variable name.

  • Index starts from 0
  • Fast access using index
  • Array name represents base address

1.1 One-Dimensional Array

Syntax


data_type array_name[size];

Example


#include <stdio.h>

int main() {
    int marks[5] = {70, 85, 90, 88, 76};

    for(int i = 0; i < 5; i++) {
        printf("%d ", marks[i]);
    }
    return 0;
}

1.2 Two-Dimensional Array

Syntax


data_type array_name[rows][columns];

Example


#include <stdio.h>

int main() {
    int matrix[2][2] = {
        {1, 2},
        {3, 4}
    };

    for(int i = 0; i < 2; i++) {
        for(int j = 0; j < 2; j++) {
            printf("%d ", matrix[i][j]);
        }
        printf("\n");
    }
    return 0;
}

2. Pointers in C

A pointer is a variable that stores the memory address of another variable. Pointers allow direct memory manipulation.

  • Used for dynamic memory allocation
  • Improves performance
  • Essential for arrays and functions

2.1 Pointer Declaration and Initialization

Syntax


data_type *pointer_name;

Example


#include <stdio.h>

int main() {
    int a = 10;
    int *ptr = &a;

    printf("Value of a: %d\n", a);
    printf("Address of a: %p\n", ptr);
    printf("Value using pointer: %d\n", *ptr);

    return 0;
}

2.2 Pointer Arithmetic

Pointer arithmetic allows moving through memory locations. Incrementing a pointer moves it by the size of its data type. 


#include <stdio.h>

int main() {
    int arr[3] = {10, 20, 30};
    int *ptr = arr;

    for(int i = 0; i < 3; i++) {
        printf("%d ", *(ptr + i));
    }
    return 0;
}

2.3 Relationship Between Arrays and Pointers

The name of an array is a pointer to its first element. Therefore, arrays and pointers are closely related. 


#include <stdio.h>

int main() {
    int arr[4] = {5, 10, 15, 20};

    for(int i = 0; i < 4; i++) {
        printf("%d ", *(arr + i));
    }
    return 0;
}

Advantages of Using Pointers

  • Efficient memory usage
  • Supports dynamic memory allocation
  • Useful for large data structures

Common Mistakes with Pointers

  • Using uninitialized pointers
  • Forgetting to free allocated memory
  • Accessing invalid memory locations
