Structures & File Handling in C

Detailed explanation of structures, nested structures, and file handling in C programming with syntax and examples.

1. Structures in C

A structure is a user-defined data type that allows grouping variables of different data types under a single name.

Syntax


struct structure_name {
    data_type member1;
    data_type member2;
};

Example


#include <stdio.h>

struct Student {
    int id;
    char name[20];
    float marks;
};

int main() {
    struct Student s1 = {101, "Rahul", 89.5};

    printf("ID: %d\n", s1.id);
    printf("Name: %s\n", s1.name);
    printf("Marks: %.2f\n", s1.marks);

    return 0;
}
  • Stores related data together
  • Improves program readability
  • Used in real-world applications

2. Nested Structure

A nested structure is a structure inside another structure. It is useful when data has a hierarchical relationship.

Syntax


struct Outer {
    data_type member1;

    struct Inner {
        data_type member2;
    } inner_object;
};

Example


#include <stdio.h>

struct Address {
    char city[20];
    int pin;
};

struct Employee {
    int emp_id;
    char name[20];
    struct Address addr;
};

int main() {
    struct Employee e1 = {1, "Anita", {"Delhi", 110001}};

    printf("ID: %d\n", e1.emp_id);
    printf("Name: %s\n", e1.name);
    printf("City: %s\n", e1.addr.city);
    printf("PIN: %d\n", e1.addr.pin);

    return 0;
}

3. File Handling in C

File handling allows data to be stored permanently in files. C provides file handling using the FILE structure.

File Pointer Declaration


FILE *fp;

File Opening Syntax


fp = fopen("filename", "mode");
  • "r" – read
  • "w" – write
  • "a" – append
  • "r+" – read & write

3.1 Writing to a File


#include <stdio.h>

int main() {
    FILE *fp;
    fp = fopen("data.txt", "w");

    fprintf(fp, "Welcome to C File Handling");

    fclose(fp);
    return 0;
}

3.2 Reading from a File


#include <stdio.h>

int main() {
    FILE *fp;
    char text[100];

    fp = fopen("data.txt", "r");

    fgets(text, 100, fp);
    printf("%s", text);

    fclose(fp);
    return 0;
}

Common File Handling Functions

  • fopen() – open file
  • fprintf() – write to file
  • fscanf() – read from file
  • fgets() – read string
  • fputc() – write character
  • fclose() – close file
